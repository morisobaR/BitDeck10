# BitDeck10

![morisoba/bitdeck10](https://imgur.com/ca8egkz.jpg)

## Overview

BitDeck10 is a 5x2 keys + 1 rotary encoder macro keyboard with OLED.

<a href="doc/readme.md">Build Guide</a> (Japanese only)

The design of the rotary encoder is based on the THQWGD001C created by Taro Hayashi.<br/>
https://github.com/Taro-Hayashi/THQWGD001
